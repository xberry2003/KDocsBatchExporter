# 项目进度

KDocsBatchExporter 当前为 `0.1.0` 阶段的结构化 Node.js 工程。

已完成：

- KDocs 目录扫描、文件身份归一化和去重。
- 普通文件与 AirPage 在线文档的类型识别及下载路由。
- AirPage `.otl` 文档官方 DOCX 导出流程。
- 直接下载器、输出路径清理、重试策略、文件格式校验和 JSONL manifest。
- 统一导出编排、扫描审计、失败清单、人工处理清单和失败项重试。
- 直接下载、AirPage 单篇和 AirPage 批量样本的 Golden 回归脚本。
- 命令行入口、配置模块及无真实账号依赖的单元测试。

当前验证结果：

- 本分支会在提交前重新运行单元测试与语法检查。

后续重点是继续扩大真实环境 Golden A/B 回归覆盖、批量任务恢复验证和更完整的集成测试。在这些验证完成前，不建议直接执行企业空间全量导出。

本仓库不包含已废弃的 RPA 方案，也不包含 Cookie、CSRF、Token、浏览器会话或真实导出数据。
